Etiquetas de Formato


Podemos asignar negritas en nuestro texto, para eso, podemos usar las etiquetas b y strong


esto es texto en negritas esto tambien es texto en negritas

Cual es la diferencia entre las dos? nuevamente, es importancia semantica. bold no tiene importancia semantica, pero strong si tiene importancia semantica


Pasa lo mismo con italic o enfasis, se puede usar dos etiquetas, i y tambien em

esto es un texto en italic esto tambien es un texto en italic

cual es la diferencia? nuevamente, importancia semantica, la etiqueta em si tiene importancia semantica


Tenemos mas etiquetas, como por ejemplo la etiqueta small que permite disminuir texto

por otro lado, con la etiqueta mark se puede marcar un extracto de linea

con la etiqueta del se puede tachar un texto

con la etiqueta ins se puede insertar texto, lo sub raya

con la etiqueta sub puedes bajar texto, como por ejemplo delimitar una base cuadrada de un logaritmo

por ultimo, con la etiqueta sup pueden crear o definir la elevacion de una potencia, como por ejemplo 2(2+x)2
