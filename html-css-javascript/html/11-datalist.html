Datalist


Datalist es una funcionalidad nueva de HTML5 que permite a los usuarios
poder crear listas de sugerencias de forma nativa y al seleccionar permite
usar un custom value

  1. datalist permite cargar una lista de sugerencias pero de forma nativa
  2. con la etiqueta datalist se crea la lista se sugerencias
  3. el input y el datalist se enlazan mediante el atributo list
  4. en el datalist, el id debe ser el mismo que en el list para funcionar
