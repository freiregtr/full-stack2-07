Tablas


las tablas nos permiten ordenar de forma estructurada datos en filas y columnas
este es un ejemplo de una tabla clasica con las siguientes etiquetas
tr significa table row y representa una fila, las filas son horizontales
td significa table data crea celdas dentro de las columnas, las columnas son verticales

Esta es una forma anticuada de hacer tablas, ya que como les explicaba previamente, existen etiquetas que ayudan a la semantica
la etiqueta thead indica la cabecera de la tabla
dentro de thead, se puede usar en vez de td, th ya que representa el indice de la tabla, de hecho, lo resalta en negrita
la etiqueta tbody indica el contenido de la tabla acorde al indice o en este caso, thead
por ultimo y no siempre usado pero si existe, la etiqueta tfoot indica el pie de una tabla
