Inputs


Email

  1. los tipos de inputs nor permiten validar la informacion que estamos
    agregando en cada tipo de campo, ya sea correo, numeros, etc
  2. el primer tipo de input se llama email y verifica del lado del front end
    si la estructura de correo es valida



Password

  1. tambien podemos usar un tipo de input llamado password
  2. la gracia que tiene es que todos los datos ingresados estaran ocultos
  3. nunca se debe enviar contrasenas mediante method="GET" en el formulario



Number

  1. para poder enviar numeros, podemos usar el input de tipo number
  2. podemos fijar minimos en el input con el atributo min
  3. podemos fijar maximos en el input con el atributo max



File

  1. Podemos usar el input de tipo file para poder enviar archivos
  2. podemos definir que tipo de archivo necesitamos, con el atributo accept y permitir por ejemplo solo archivos jpg o png accept=".jpg,.png"
  3. por otro lado si se necesita multiples archivos, se agrega la etiqueta multiple



Rango

  1. podemos usar un rango de como tipo de input, donde por ejemplo, podemos usarlo como rango de precios
  2. el tipo de input se llama range
  3. podemos setear un min, max y un value pre configurado



Colores

  1. Podemos seleccionar colores mediante el tipo de input color



Fecha y hora

  1. tambien podemos seleccionar fechas con el input de tipo date
  2. con value podemos definir una fecha custom
  3. podemos al igual que numbers setear un min y max
  4. es importante respetar el orden americano de fechas, primero AAAA-MM-DD

  1. tambien no solo podemos seleccionar fecha, si no tambien hora con tipo de inputdatetime-local



  1. tambien podemos seleccionar solo la hora con el input type time



Hideen

  1. tambien podemos dejar inputs ocultos con hidden con el input type hidden
  2. esto es util cuando queremos procesar datos pero no queremos que el usuario interactue con el



Reset y boton enviar

  1. dentro de los controles de html y formularios podemos reiniciar un formulario con el boton
    reset al mismo tiempo de tener mas de una forma de enviar el formulario
    con la etiqueta button
