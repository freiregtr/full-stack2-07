Imagenes


para cargar imagenes, se debe llamar a la etiqueta img dentro de studio code
en este punto, el primer atributo que aparece se llama src y este especifica la ruta interna
para nuestras pruebas, tenemos una imagen cargada en la siguiente ruta en vscode /img/aws_imagenes.jpg

imagen de ejemplo


tenemos diferentes atributos
con el atributo width podemos ajustar el ancho de una foto
con el atributo height podemos ajustar el alto de una fotografia
con el atributo title podemos agregar una leyenda a la foto
con el atributo alt podemos agregar un texto alternativo en caso que la imagen no exista, este texto se mostrara SIEMPRE Y CUANDO
la imagen no cargue, ademas, sirve para sistemas para personas no videntes, asi estos sistemas describen las imagenes a las personas no videntes
