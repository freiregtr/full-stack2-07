Formularios


Los formularios permiten enviar un correo o un mensaje al administrador del sitio
del sitio, existen varias etiquetas y atributos para usarlos con formularios

  1. para agregar campos, se usa la etiqueta input
  2. para agregar un texto a ese campo, se usa label
  3. con el atributo for se asocia el label con input mediante el id
  4. para asociar el input con el for, se debe sincronizar mediante el id

  1. Para agregar un contexto de que agregar, usamos placeholder

  1. Por otro lado, el atributo maxlength permite fijar un maximo de caracteres

  1. para agregar una descripcion o mas texto, se usa el marcador textarea
  2. con el atributo name le indicamos mediante clave-valor
    que dentro del nombre name, vendra el contenido del mensaje

  1. tambien podemos seleccionar cosas con la etiqueta select
  2. con la etiqueta option podemos agregar diferentes opciones

  1. Tambien podemos seleccionar diferentes opciones mediante radio button
  2. con radio podemos seleccionar simplemente pinchando una opcion
  3. para que pueda el servidor backend sepa que dato se esta enviando, debe usarse
    el atributo value asi llega mediante clave-valor al server
  4. para que de un grupo de mas opciones podamos seleccionar solo uno, debemos
    usar el atrinuto name asi se podra seleccionar solo uno de un grupo

  1. ademas, para enviar datos true o false, podemos usar etiquetas
    como checkbox

  1. por ultimo, debemos tener un boton para enviar el formulario
    con el atributo de tipo submit
  2. si no ponen nada, automaticamente va a rellenar con Enviar consulta
  3. Para agregar un nombre al boton, usamos el atributo value
